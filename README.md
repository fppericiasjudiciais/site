# FP Pericias Judiciais

Site institucional estatico de Fernando Alves Pires Neto para divulgacao de servicos de pericia judicial e assistencia tecnica.

## Estrutura

- `index.html`: pagina inicial
- `sobre.html`: perfil profissional
- `especialidades.html`: areas de atuacao
- `contato.html`: canais de contato e formulario com envio para WhatsApp
- `assets/css/`: estilos do site
- `assets/js/contact.js`: montagem da mensagem do formulario para WhatsApp
- `assets/images/`: imagens e assets visuais

## Publicacao no GitHub Pages

1. Envie os arquivos para o repositorio no GitHub.
2. No GitHub, abra `Settings`.
3. Entre em `Pages`.
4. Em `Build and deployment`, selecione `Deploy from a branch`.
5. Escolha a branch `main` e a pasta `/root`.
6. Salve e aguarde a URL de publicacao.

Depois que a URL estiver disponivel, atualize os metadados sociais das paginas com o endereco final do site.
